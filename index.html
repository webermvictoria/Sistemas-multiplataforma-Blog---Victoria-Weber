Publicaciones del blog

La importancia de la tecnología

La tecnología forma parte de la vida cotidiana y permite resolver problemas, comunicarnos y acceder a información de manera rápida.

Aprender programación

Programar ayuda a desarrollar el pensamiento lógico. En este proyecto se usan HTML, CSS, JavaScript y Bootstrap para construir una aplicación simple y clara.

Aplicaciones multiplataforma

Una aplicación multiplataforma debe verse correctamente en computadoras, tablets y celulares. Por eso se aplicó un diseño responsive.

Sobre el proyecto

Este sitio fue realizado con una estructura sencilla para demostrar navegación interna, organización de contenido y diseño adaptable. Su diseño fue inspirado en un proyecto personal llamado Ágora, realizado para un taller de proyectos.

Incluye:

  • Página de bienvenida
  • Sección tipo blog
  • Menú de navegación
  • Header y footer
  • Diseño responsive

Información del trabajo

Tecnologías utilizadas

Para desarrollar esta aplicación se utilizó HTML para la estructura, CSS para los estilos, JavaScript para cambiar entre las páginas del sitio sin recargar el navegador y Bootstrap como biblioteca de apoyo para el diseño web.

Desafíos encontrados

Uno de los desafíos fue ordenar el contenido para que se vea bien en distintos tamaños de pantalla. También se buscó que el código sea claro y fácil de modificar.
